# Explainable Machine Learning Course Materials & Projects
This repository contains my course materials and projects for the Explainable Machine Learning course at UW-Madison taught by Prof. Kris Sankaran. It is forked from the [official course repository](https://github.com/krisrs1128/stat992_s25)

## Repository Structure
- `notes`: Notes provided by the instructor.
- `demos`: The demos for the course provided by the instructor.
- `exercises` & `logistics`: The exercises instruction and course logistics provided by the instructor.
- `Mynotes`: My weekly notes for the course.
